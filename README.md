# Android-TicTacToe

Android-TicTacToe is a tic-tac-toe game that was built using Android Studio with Java programming language.

## Demo

![Game Demo](https://media.giphy.com/media/jpKpsHKmqqXiZPlvn4/giphy.gif)

## Installation

In order to play the game, you must do the following to install Android Studio.

1. [Link to install Android Studio](https://www.youtube.com/watch?v=Z98hXV9GmzY&feature=emb_logo)

2. [Link to create Android Virtual Device (AVD) emulator to play the game](https://www.youtube.com/watch?v=LgRRmgfrFQM)
